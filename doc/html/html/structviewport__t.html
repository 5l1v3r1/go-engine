Detailed Description

Estructura para representar un "viewport" o una fracción de pantalla.

http://en.wikipedia.org/wiki/Viewport

Un viewport no es más que una abstracción de una "ventana" o portal desde la ventana de nuestro programa hasta el mundo en 3D de la aplicación. Básicamente, es una porción de pantalla (definida por una posición y un tamaño) en la que se dibujarán ciertos elementos del juego.

En una ventana, un viewport se define más o menos así:

 ______________________________
|                              |
|      x,y_____________        |
|        |             |       |
|        |             |height |
|        |_____________|       |
|             width            |
|______________________________|

Para más información, véase CComponent_Camera y CSystem_Render.
