Descripción detallada

Vector tridimensional.

Estructura que representa un vector de flotantes con 3 componentes:

(x, y, z)
*

Véase http://es.wikipedia.org/wiki/Vector

Con esta clase, se puede hace gran variedad de operaciones. Para una mayor variedad, véase la clase CSystem_Math.

GLfloat vector3f_t::precision = 0.01f
static

Precisión.

Se utiliza para comparar si 2 puntos son equivalentes, dada una precisión decimal para cada componente. Así, si el valor absoluto de la diferencia entre 2 vectores es menor que la precisión para cada componente, se dice que son iguales.
