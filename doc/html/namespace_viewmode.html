Descripción detallada

Modos de vista o viewmodes.

Espacio de nombres usado para guardar las constantes y valores de los distintos tipos de modos de vista (perspectiva, ortografica, ortografica a tamaño de pantalla, etc).

Documentación de las enumeraciones

enum Viewmode::viewmodes_t

Enum de modos de vista.

Cada modo de vista tiene un valor asociado. Principalmente, 3 modos de vista:

  • Perspectiva: Se dibujará con una proyección en perspectiva.
  • Ortográfica: Se dibujará con una proyección ortográfica. Los rangos serán [0, 1] para el eje vertical (de abajo a arriba), y [0, 1] para el eje horizontal (de derecha a izquierda).
  • ortográfica a tamaño de pantalla: Se dibujará con una proyección ortográfica. Los rangos serán [0, ALTO_VENTANA] para el eje vertical (de abajo a arriba), y [0, ANCHO_VENTANA] para el eje horizontal (de derecha a izquierda).

const char * Viewmode::viewmode_s = {"perspective", "ortho", "ortho_screen", "not_defined"}

Nombres de viewmodes.

Array de strings que contienen, por cada enum de viewmodes_t, una representación con caracteres. La representación como cadena de caracteres será idéntica al nombre del valor del enum, salvo para el caso especial *__viewmode_not_defined*, que será *"not_defined"*.
