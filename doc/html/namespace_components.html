Descripción detallada

Componentes.

Espacio de nombres para guardar enumeraciones, variables y métodos referentes a los componentes.

Atención
Esto debería en otro sitio más generico, no en el fichero de definición del componente abstracto.

Documentación de las enumeraciones

enum Components::components_t

Enumeración o identificadores de componentes.

Enum para identificar a todos los tipos de componentes. Cada clase heredada de CComponent debe tener un valor en esta enumeración.

components_t Components::string_to_component ( const std::string &  c)

Transformar un string a un enum de un componente.

Dado un nombre, trata de transformar dicho nombre a una enum de componentes.

Parámetros

Devuelve
En caso de encontrar un componente, devuelve su valor de enum. En caso contrario, devuelve components_t.__component_not_defined.

Nombres de componentes.

Array de strings que contienen, por cada enum de components_t, una representación con caracteres. La representación como cadena de caracteres será idéntica al nombre del valor del enum, salvo para el caso especial *__component_not_defined*, que será *"not_defined"*.
