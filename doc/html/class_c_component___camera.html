Descripción detallada

Componente cámara.

Dado un escenario tridimensional en nuestro programa, es necesario crear una manera de observar lo que ocurre dentro de dicho escenario. El objetivo principal de este componente es actuar como una ventana a nuestro mundo. Esto es, observar el espacio y dibujarlo en la ventana de nuestro programa.

A este proceso se le conoce como renderizado. Cada objeto es dibujado a su manera, bajo sus componentes y sus funciones de dibujado. Por ello, la cámara se encargará de dibujar el mundo tal y como es percibido por ella. Por tanto, asignada una cámara a un objeto, bajo las propiedades del componente CComponent_Transform, se dibujará, según la posición y la orientación de la cámara, el mundo tal y como es observado.

No obstante, existen varios tipos de cámara, enumerados en Viewmode::viewmodes_t. Dependiendo del tipo de vista, el escenario se dibujará de una forma o de otra.

Además, cabe destacar que se puede escoger una porición de la pantalla sobre la que dibujará la cámara. Esta será nuestra ventana o viewport, que no será más que un rectángulo dentro de nuestra ventana.

Además, toda cámara debe estar registrada en el sistema CSystem_Render para poder escribir datos en pantalla.

Atención
No se eliminará el objeto de la lista de cámaras. Deberá hacerse de manera manual desde el sistema CSystem_Render.

Por ejemplo, creamos y registramos una cámara de la siguiente manera:

CGameObject* camara_main = gGameObjects.AddGameObject("camara_main");
camara_main->Transform()->position.y = -3.f;
camara_main->Transform()->position.z = -10.f;
camara_main->Camera()->far_clip = 500.f;
camara_main->Preserve(); // Preservar entre escenas.
camara_main->SetInputFunction(&Camara_main_movimiento); // Cambiar el callback por una función dada.
gRender.AddCamera(camara_main); // Registrar en el sistema render.
camara_main->Camera()->skybox_texture = "skybox1";

Finalmente, cabe destacar que la cámara puede definir un color o un cielo (skybox) de fondo, que servirá para dibujar contenido en el fondo de nuestro escenario.

bool CComponent_Camera::clear
